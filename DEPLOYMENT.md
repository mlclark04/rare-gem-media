# Deployment Checklist

## Before You Push to GitHub

- [ ] All HTML files are in the root directory
- [ ] File names are correct: `index.html`, `pricing.html`, `about.html`, `contact.html`
- [ ] All internal links work (test locally first if possible)
- [ ] Email address in contact page is correct
- [ ] You have your Formspree form ID ready

## Setting Up GitHub Pages

- [ ] Repository is **Public** (not Private)
- [ ] Go to Settings → Pages
- [ ] Source is set to "main branch"
- [ ] Wait 1–2 minutes for GitHub to deploy
- [ ] Test your site at `https://YOUR_USERNAME.github.io/rare-gem-media/`

## After Going Live

- [ ] Test all page links (home → services → pricing → about → contact)
- [ ] Fill out the contact form and verify it works
- [ ] Check that all images/graphics load correctly
- [ ] Test on mobile (use Chrome DevTools)
- [ ] Share your live URL!

## Optional: Custom Domain Setup

- [ ] Domain registered (e.g., raregemmedia.com)
- [ ] DNS records added to your registrar
- [ ] Custom domain entered in GitHub Pages settings
- [ ] HTTPS enforcement enabled
- [ ] Wait up to 48 hours for DNS to propagate
- [ ] Verify site loads at your custom domain

## Ongoing

- [ ] Keep a local backup of all HTML files
- [ ] Test changes locally before pushing to GitHub
- [ ] Monitor contact form submissions
- [ ] Update content as needed (push to GitHub to deploy)
